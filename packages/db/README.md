# @hawktask/db

Supabase types for Hawktask. Shared database types for users, clients, proposals, and events.

- **src/types.ts** – TypeScript interfaces aligned with the schema in the PRD.
- **src/index.ts** – Re-exports for `@hawktask/db`.

Use `npx supabase gen types typescript` and replace or extend these types when the Supabase project is set up.
